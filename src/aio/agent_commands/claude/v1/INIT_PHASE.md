# AIO Init Phase — Claude

Collect the presentation brief from the user.

Ask for:
1. **Topic**: What is this presentation about?
2. **Audience**: Who will see it? (technical, executive, general)
3. **Slide count**: How many slides? (recommended: 8–15)
4. **Theme**: Which theme? (minimal, modern, vibrant, or custom)
5. **Key points**: 3–5 main messages

Output a structured brief confirming the above before proceeding to composition.
