# AIO Generate Command

Generate a complete slides.md for:

**Topic**: {topic}
**Slide count**: {slides_count}
**Theme**: {theme}
**Agent**: {agent}

## Instructions

Produce a complete slides.md file with:
- YAML frontmatter (title, agent, theme)
- Each slide separated by `---`
- `<!-- @layout: layout-name -->` on each slide
- Substantive content (not placeholder text)
- Progressive narrative from introduction to conclusion
