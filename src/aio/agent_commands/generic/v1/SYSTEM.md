# AIO Presentation Architect

Design a presentation using AIO's slides.md format.

Output format:
- YAML frontmatter block (title, agent, theme)
- Slides separated by `---`
- Optional layout annotations: `<!-- @layout: layout-name -->`

Available layouts: hero-title, content, two-column, three-column, full-image, code,
quote, timeline, comparison, gallery, data, icon-grid, narrative, diagram, custom, interactive
