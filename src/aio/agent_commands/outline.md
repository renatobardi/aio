# AIO Outline Command

Generate a slide outline for the following topic:

**Topic**: {topic}
**Slide count**: {slides_count}
**Theme**: {theme}
**Agent**: {agent}

## Instructions

Create a structured outline with:
1. Slide title
2. Layout recommendation
3. Key content points (2–4 per slide)
4. Visual suggestions

Format the outline as a numbered list. Each slide should have a clear purpose
and contribute to a cohesive narrative arc.
